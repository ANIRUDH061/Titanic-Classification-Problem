## Titanic Survival Prediction
Welcome to my Titanic Survival Prediction project! This repository contains a machine learning model designed to analyze passenger data and predict whether a passenger survived the Titanic shipwreck.

# Problem Statement
The goal of this project is to build a binary classification model that predicts survival outcomes based on passenger attributes. By analyzing features such as age, gender, ticket class, and family size, we aim to understand the key factors that influenced survival during the disaster.

# Tech Stack
This project was built using Python and the following libraries:

Pandas: For data manipulation and analysis.

NumPy: For numerical computations.

Seaborn: To load the dataset and visualize relationships.

Scikit-Learn: For data splitting, model training (Logistic Regression), and evaluation.

# Project Workflow
Data Loading: Used the built-in seaborn Titanic dataset.

Data Cleaning:

Handled missing values (e.g., filling missing ages with the mean).

Converted categorical data (gender) into numerical format.

Model Training: Implemented a Logistic Regression model, a robust choice for binary classification tasks.

Evaluation: Assessed the model's performance using Accuracy Score, Confusion Matrix, and a Classification Report.

# Conclusion
This project successfully implemented a Logistic Regression model to predict Titanic passenger survival, achieving an accuracy of approximately 78%. The analysis confirms that key features—specifically gender, passenger class, and age—were the most significant determinants of survival. This project serves as a clear, introductory example of how data preprocessing and classification algorithms can be used to extract meaningful insights from historical data.
